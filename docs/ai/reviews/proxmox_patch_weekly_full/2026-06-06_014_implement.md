## 変更内容の要約

### 1. Step 2b の NO_UPDATES ブロックからメール送信を削除（Medium 対応）

`proxmox_patch_dryrun` role が `NO_UPDATES` の場合にすでに `[Proxmox Patch Dry-run] NO_UPDATES` メールを送信する。
Step 2b でも独自に NO_UPDATES メールを送っていたため、1回の実行で2通届いていた。

変更前:

```yaml
- name: Notify NO_UPDATES and end playbook cleanly
  when: _status == 'NO_UPDATES'
  block:
    - name: Load mail variables
      ansible.builtin.include_vars: ...
    - name: Send NO_UPDATES notification
      community.general.mail: ...
    - name: Set NO_UPDATES skip flag for subsequent plays
      ansible.builtin.set_fact:
        _weekly_full_no_updates: true
    - name: End this play
      ansible.builtin.meta: end_play
```

変更後:

```yaml
- name: Notify NO_UPDATES and end playbook cleanly
  when: _status == 'NO_UPDATES'
  block:
    - name: Set NO_UPDATES skip flag for subsequent plays
      ansible.builtin.set_fact:
        _weekly_full_no_updates: true
    - name: End this play (NO_UPDATES - normal exit, notification sent by dry-run role)
      ansible.builtin.meta: end_play
```

NO_UPDATES の通知は dry-run role に一本化した。Step 2b は skip flag のセットと `meta: end_play` のみを行う。

### 2. Steps 5・9 の gather_facts を false に変更し setup を明示追加（Low 対応）

Ansible の fact gathering は tasks より先に実行される。`gather_facts: true` のままでは、NO_UPDATES 経路で最初のタスクが `meta: end_play` であっても、その前に pve2/pve1 への SSH 接続と fact 収集が発生していた。

`gather_facts: false` に変更し、skip guard (`meta: end_play`) の通過後に `ansible.builtin.setup` を明示的に呼ぶことで fact を収集する。

変更前（Steps 5・9 共通）:

```yaml
hosts: pveN
gather_facts: true
tasks:
  - name: Skip if NO_UPDATES
    ansible.builtin.meta: end_play
    when: hostvars['localhost']['_weekly_full_no_updates'] | default(false)

  - name: Run healthcheck on pveN
    ansible.builtin.include_role:
      name: proxmox_healthcheck
```

変更後（Steps 5・9 共通）:

```yaml
hosts: pveN
gather_facts: false
tasks:
  - name: Skip if NO_UPDATES
    ansible.builtin.meta: end_play
    when: hostvars['localhost']['_weekly_full_no_updates'] | default(false)

  - name: Gather facts for healthcheck role
    ansible.builtin.setup:

  - name: Run healthcheck on pveN
    ansible.builtin.include_role:
      name: proxmox_healthcheck
```
