## 変更内容

`scripts/codex-classify.sh` の Step 4 を以下のように変更した。

変更前:

```bash
codex exec "$(cat "$PROMPT_FILE")" > "$CODEX_OUTPUT" 2>&1
```

変更後:

```bash
if ! codex exec "$(cat "$PROMPT_FILE")" > "$CODEX_OUTPUT" 2>&1; then
    echo "Error: codex exec failed" >&2
    sed -n '1,200p' "$CODEX_OUTPUT" >&2
    exit 1
fi
```

`codex exec` が非 0 で終了した場合に `$CODEX_OUTPUT` の先頭 200 行を stderr に出力してから `exit 1` する。
`$CODEX_OUTPUT` の削除は `trap cleanup EXIT` に委ねており、ログ出力後も確実に削除される。
