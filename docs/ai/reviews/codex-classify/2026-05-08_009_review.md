該当箇所:

- `scripts/codex-classify.sh:132-136`

```bash
if ! codex exec "$(cat "$PROMPT_FILE")" > "$CODEX_OUTPUT" 2>&1; then
    echo "Error: codex exec failed" >&2
    sed -n '1,200p' "$CODEX_OUTPUT" >&2
    exit 1
fi
```

これにより、`set -e` と `trap cleanup EXIT` により Codex CLI の失敗詳細が消える問題は解消されている。
