### M3: regex_replace の二重置換

**対応内容:**

`tasks/main.yml` 内の 3 箇所を `'(.*)'` から `'^(.*)$'` に変更:
- ZFS pool メッセージ生成
- replication stale (>24h) メッセージ生成
- replication stale (>2h) メッセージ生成

Python の `re.sub('(.*)', ...)` は末尾の空文字列にも再度マッチするため、`'^(.*)$'` で行全体にアンカーして二重置換を防ぐ。

**変更ファイル:**
- `roles/proxmox_healthcheck/tasks/main.yml`

### M4: 閾値の境界表現を requirement doc に合わせる
