# proxmox_patch_apply_node 追加要求仕様（apply_mode 追加）

作成日: 2026-05-09

---

## 変更内容

### 1. roles/proxmox_patch_apply_node/defaults/main.yml

以下を追加する:

```yaml
proxmox_patch_apply_mode: "auto"
# auto: MAINTENANCE_REQUIRED 検出で abort
# manual: MAINTENANCE_REQUIRED 検出でも続行
```

### 2. roles/proxmox_patch_apply_node/tasks/main.yml — Phase 2 判定修正

MAINTENANCE_REQUIRED 検出時の動作を apply_mode によって分岐する:

```
apply_mode == "auto"   → abort（メール送信して終了）
apply_mode == "manual" → 警告メッセージを出して続行
```

## 使い方

```bash
# PATCH_READY 自動適用（デフォルト）
ansible-playbook -i inventories/homelab/hosts.yml playbooks/proxmox_patch_apply_node.yml \
  -e target_node=pve2

# MAINTENANCE_REQUIRED 手動適用
ansible-playbook -i inventories/homelab/hosts.yml playbooks/proxmox_patch_apply_node.yml \
  -e target_node=pve2 \
  -e proxmox_patch_apply_mode=manual
```
