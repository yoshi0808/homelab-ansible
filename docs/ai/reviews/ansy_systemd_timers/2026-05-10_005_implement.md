# 追加実装内容: connection: local 追加

## 変更概要

ansy 上で playbook を実行した際、ansy.internal への SSH 接続を試みて失敗する問題が発生した。
ansy から ansy 自身を対象とする playbook のため、SSH 接続は不要。
`connection: local` を追加してローカル処理に切り替えた。

## 変更ファイル

| ファイル | 変更内容 |
|---|---|
| `playbooks/ansy_systemd_timers.yml` | `connection: local` を追加 |

## 変更差分

```yaml
# 変更前
- name: Configure Ansible systemd timers on ansy
  hosts: dev_nodes
  gather_facts: false
  become: true

# 変更後
- name: Configure Ansible systemd timers on ansy
  hosts: dev_nodes
  gather_facts: false
  become: true
  connection: local
```

## 確認コマンド・結果

```
ansible-playbook --syntax-check playbooks/ansy_systemd_timers.yml -i inventories/homelab/hosts.yml
```

```
playbook: playbooks/ansy_systemd_timers.yml
```

syntax-check パス。エラーなし。
