## 実装概要

`reports_base_dir` を `inventories/homelab/group_vars/all.yml` に絶対パスで定義し、
全 role の `defaults/main.yml` で `playbook_dir/../reports/...` を
`reports_base_dir/...` に置き換えた。

## 注意点

- `reports_base_dir` は `all.yml` で定義するため、全グループ・全ホストに適用される。
  Semaphore 実行時も quory CLI 実行時も同じパスが使われる。
- `reports/` ディレクトリ自体の作成は各 role の tasks 側（`file: state=directory`）が担っており、本変更では触れていない。
- Semaphore 側で別のパスを使いたい場合は、Semaphore の Extra Variables で `reports_base_dir` を上書きすることで対応可能。
