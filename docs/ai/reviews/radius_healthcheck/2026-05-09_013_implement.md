# radius_healthcheck 追加実装: body の for ループ書き換え

実装日: 2026-05-09

---

## 変更内容

**変更ファイル:** `roles/radius_healthcheck/tasks/main.yml`

`Send healthcheck notification` の `body` 内、Criticals と Warnings の `join('\n')` を `{% for item in ... %}` ループに書き換えた。

変更前:
```yaml
    body: |
      Host    : {{ inventory_hostname }}
      Status  : {{ radius_report.result.status }}

      Criticals:
      {{ radius_criticals | join('\n') if radius_criticals | length > 0 else '(none)' }}

      Warnings:
      {{ radius_warnings | join('\n') if radius_warnings | length > 0 else '(none)' }}
```

変更後:
```yaml
    body: |
      Host    : {{ inventory_hostname }}
      Status  : {{ radius_report.result.status }}

      Criticals:
      {% for item in radius_criticals %}
      - {{ item }}
      {% else %}
      (none)
      {% endfor %}
      Warnings:
      {% for item in radius_warnings %}
      - {{ item }}
      {% else %}
      (none)
      {% endfor %}
```

Jinja2 の `for ... else ... endfor` 構文を使い、リストが空の場合は `else` 節の `(none)` を出力する。
